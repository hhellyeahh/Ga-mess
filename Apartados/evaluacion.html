· Evaluación ·

La evaluación final y el cierre del proyecto son etapas cruciales para garantizar su éxito
y sacar conclusiones importantes para futuros proyectos. En nuestra empresa de desarrollo
de aplicaciones de noticias de videojuegos, estas fases serán fundamentales para consolidar
nuestro trabajo y aprender de la experiencia adquirida.


Evaluación Final

En la evaluación final, nos enfocaremos en medir la eficacia y eficiencia del proyecto. Para ello,
evaluaremos el cumplimiento de los objetivos establecidos y analizaremos cualquier desviación que
haya ocurrido durante su desarrollo. También evaluaremos la gestión del proyecto y sus procesos,
identificando tanto los éxitos como los fracasos.

Se llevarán a cabo diversas pruebas para garantizar el cumplimiento de la funcionalidad requerida.
Se realizan pruebas exhaustivas para asegurar que la aplicación realice todas las operaciones y
acciones de manera correcta.

Además, se prestará especial atención a detectar cualquier desviación que pueda haber ocurrido
durante el desarrollo del proyecto. En caso de identificar problemas, se realizará un análisis
detallado para comprender las causas y se buscarán opciones de mejora.


Las pruebas que se llevarán a cabo incluirán:

- Test unitarios: Después de completar cada tarea, se enviarán las funcionalidades
correspondientes al equipo de prueba (tester) para realizar pruebas exhaustivas. Si se detectan
fallos, se devolverán los elementos al equipo de desarrollo con las áreas identificadas para su
corrección.

- Test integrados: Una vez finalizadas las tareas y realizadas las pruebas unitarias,
se llevará a cabo un test integrado del proyecto. En caso de fallos, se volverá a realizar
pruebas unitarias para identificar y solucionar los problemas.

- Pruebas de usuario: Después de completar el desarrollo y las pruebas de código, se
llevará a cabo una prueba de usuario para identificar posibles defectos en la accesibilidad
y visibilidad del programa.

- Pruebas del cliente: Finalmente, se invitará a agentes externos a la empresa a realizar
pruebas finales del proyecto. El objetivo es recopilar comentarios para verificar si el proyecto
cumple con las expectativas. En caso de identificar áreas de mejora, se retrocederá a pasos
anteriores para implementar las mejoras necesarias.

Estas pruebas son fundamentales para garantizar la calidad y el éxito del proyecto, así como para
asegurar la satisfacción del cliente y usuarios finales.


Cierre del Proyecto

Una vez completada la evaluación, procederemos al cierre del proyecto. Entregaremos el producto
final y nos aseguraremos de que los usuarios estén satisfechos con el resultado. Generamos una
memoria técnica detallada del producto o servicio desarrollado y analizaremos el alcance de los
objetivos, así como las desviaciones ocurridas durante el proyecto.

Además, realizaremos una justificación económica detallada de los gastos realizados durante el proyecto.
Escribiremos un informe final que recopile todo lo sucedido durante el proyecto para futuras referencias.
Con estas acciones, aseguraremos un cierre adecuado del proyecto y estaremos listos para embarcarnos en
futuras iniciativas con mayor experiencia y conocimiento acumulado.
